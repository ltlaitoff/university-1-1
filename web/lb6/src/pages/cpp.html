Скорочена довідка

C++ - об'єктно-орієнтована мова програмування, розроблена в 1979 році Бьорном Страуструпом. Вона поєднує в собі можливості C і C++ , дозволяючи працювати з високорівневими та низькорівневими конструкціями.

Особливості

  • Об'єктно-орієнтованість Підтримує класи, об'єкти, спадкування, поліморфізм та інкапсуляцію.
  • Шаблони Дозволяють написати загальний алгоритм, який працює з різними типами даних.
  • Низькорівневий доступ Дозволяє працювати з пам'яттю та адресами.
  • Багатопоточність Має бібліотеки для роботи з потоками виконання.
  • Стандартна бібліотека Включає в себе багатий набір готових класів та функцій для швидкого програмування.

Приклад програми (Hello World)

#include < iostream>
using namespace std;

int main() {
	cout << "Hello World!";
	return 0;
}

Хронологія розвитку

  • 1979 Створення мови C++ Бьорном Страуструпом
  • 1983 Перша публікація книги The C++ Programming Language
  • 1998 Прийняття міжнародного стандарту C++98
  • 2011 Видання стандарту C++11 з важливими розширеннями мови
  • 2014 Видання стандарту C++14 який вносить дрібні зміни та поліпшення
  • 2017 Видання стандарту C++17 з новими функціями та бібліотеками

Переваги та недоліки

Переваги

  • Множинність парадигм Підтримка об'єктно-орієнтованого та процедурного програмування.
  • Висока продуктивність Низькорівневий доступ до пам'яті та швидке виконання.
  • Багатий функціонал Розширена стандартна бібліотека.
  • Широкі можливості оптимізації Гнучкість у виборі алгоритмів та структур даних.

Недоліки:

  • Складність вивчення Мова може бути складною для новачків.
  • Можливість низькорівневих помилок Помилки в роботі з пам'яттю можуть призвести до непередбачуваного поведінки програми.
  • Відсутність автоматичного сміття Ручне управління пам'яттю може бути важким завданням.
  • Обмежена підтримка деяких сучасних парадигм програмування Не так гнучко підходить до функціонального програмування, як деякі інші мови.
